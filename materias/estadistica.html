Estadística

Métodos estadísticos aplicados a la economía

  1. Inicio
  2. Estadística

  • Media (promedio): Suma de todos los valores dividida entre la cantidad
  • • Fórmula: μ = Σxi / n
  • • Ejemplo: Notas 7, 8, 9, 10 → Media = (7+8+9+10)/4 = 8.5
  • Mediana: Valor central cuando los datos están ordenados
  • • Datos 1, 3, 5, 7, 9 → Mediana = 5
  • • Datos 1, 3, 5, 7 → Mediana = (3+5)/2 = 4
  • • Más robusta que la media ante valores extremos
  • Moda: Valor que más se repite
  • • Datos 2, 3, 3, 4, 5 → Moda = 3
  • • Puede haber varias modas (bimodal, trimodal)
  • ¿Cuándo usar cuál?
  • • Media: datos simétricos sin outliers
  • • Mediana: datos con valores extremos (ingresos, precios de casas)
  • • Moda: datos categóricos (color favorito)
  • Media ponderada: Cada valor tiene un peso distinto según su importancia
  • • Fórmula: x̄ = Σ(wᵢ · xᵢ) / Σwᵢ
  • Media geométrica: Útil para tasas de crecimiento y rendimientos
  • • Fórmula: G = (∏xᵢ)^(1/n)
  • Percentiles y cuartiles: Dividen los datos en 100 o 4 partes iguales
  • • Q₂ coincide con la mediana; rango intercuartílico (IQR = Q₃ − Q₁) mide dispersión

  • Rango: Diferencia entre el valor máximo y mínimo
  • • Datos 2, 5, 8, 12 → Rango = 12 - 2 = 10
  • Varianza: Promedio de las desviaciones al cuadrado
  • • Fórmula: σ² = Σ(xi - μ)² / n
  • Desviación estándar: Raíz cuadrada de la varianza
  • • σ = √σ²
  • • Misma unidad que los datos originales
  • Ejemplo: Ingresos mensuales: $2000, $2500, $3000, $3500, $4000
  • • Media = $3000
  • • Desviación estándar ≈ $707
  • Coeficiente de variación: CV = (σ/μ) × 100%
  • • Permite comparar dispersión entre datasets diferentes
  • Regla empírica: En distribución normal:
  • • 68% de datos dentro de μ ± σ
  • • 95% dentro de μ ± 2σ
  • • 99.7% dentro de μ ± 3σ
  • Rango intercuartílico (IQR): Diferencia entre Q₃ y Q₁
  • • Resistente a outliers; base del diagrama de caja y bigotes
  • Desviación media absoluta: MAD = Σ|xi − μ| / n
  • • Menos sensible a valores extremos que la varianza
  • Asimetría y curtosis: Forma de la distribución
  • • Asimetría positiva (cola derecha) vs negativa (cola izquierda)
  • • Curtosis mide el grosor de las colas (leptocúrtica, platicúrtica)

  • ¿Qué es? Posibilidad de que ocurra un evento (0 a 1, o 0% a 100%)
  • Axiomas:
  • • P(A) ≥ 0 siempre
  • • P(espacio muestral) = 1
  • • Eventos mutuamente excluyentes: P(A o B) = P(A) + P(B)
  • Probabilidad condicional:
  • • P(A|B) = P(A y B) / P(B)
  • • "Probabilidad de A, dado que ocurrió B"
  • Teorema de Bayes:
  • • P(A|B) = P(B|A) × P(A) / P(B)
  • • Actualiza probabilidades con nueva información
  • Ejemplo:
  • • 10% tiene enfermedad (P(E) = 0.1)
  • • Test detecta 90% de enfermos (P(+|E) = 0.9)
  • • 5% falsos positivos (P(+|no E) = 0.05)
  • • Si test es positivo, ¿probabilidad real de estar enfermo?
  • • P(E|+) = (0.9 × 0.1) / [(0.9×0.1) + (0.05×0.9)] = 67%
  • Independencia estadística: P(A ∩ B) = P(A) × P(B)
  • • Si se cumple, A y B son independientes; saber uno no informa sobre el otro
  • Ley de probabilidad total: Particionar el espacio muestral para calcular P(B)
  • • P(B) = Σ P(B|Aᵢ) · P(Aᵢ)
  • Variable aleatoria: Función que asigna un número a cada resultado
  • • Discreta vs continua; función de probabilidad (fmp) y densidad (fdp)
  • Valor esperado y varianza: E[X] = Σ x·P(x); Var(X) = E[(X−μ)²]
  • • Propiedades de linealidad de la esperanza: E[aX+b] = aE[X]+b

  • Distribución binomial: Éxitos en n ensayos independientes
  • • P(X=k) = C(n,k) × p^k × (1-p)^(n-k)
  • • Ejemplo: Lanzar moneda 10 veces, ¿cuántas caras?
  • Distribución normal (Gauss):
  • • Forma de campana simétrica
  • • Definida por media (μ) y desviación estándar (σ)
  • • Estándar: μ=0, σ=1
  • • Muchos fenómenos naturales la siguen
  • Distribución de Poisson:
  • • Eventos raros en un intervalo de tiempo/espacio
  • • P(X=k) = (λ^k × e^-λ) / k!
  • • Ejemplo: Llamadas telefónicas por hora, accidentes por día
  • Distribución uniforme:
  • • Todos los valores igual de probables
  • • Ejemplo: Dado justos, número aleatorio
  • Distribución t de Student: Colas más anchas que la normal
  • • Grados de libertad (gl); converge a normal cuando gl → ∞
  • • Ideal para muestras pequeñas y varianza poblacional desconocida
  • Distribución chi-cuadrado (χ²): Suma de normales estándar al cuadrado
  • • Usada en pruebas de independencia, bondad de ajuste, y varianza
  • Distribución F de Fisher: Cociente de dos chi-cuadrado independientes
  • • Fundamental en ANOVA y comparación de varianzas entre grupos
  • Teorema del límite central: La media muestral tiende a normal aunque la población no lo sea
  • • Justifica usar la normal para inferencia con muestras grandes (n ≥ 30)

  • ¿Por qué muestrear? Censar toda la población es costoso/imposible
  • Muestreo aleatorio simple: Cada individuo tiene igual probabilidad
  • Muestreo estratificado: Dividir en grupos (estratos) y muestrear cada uno
  • Muestreo por conglomerados: Muestrear grupos completos
  • Tamaño de muestra:
  • • n = (Z² × σ²) / E²
  • • Z = valor Z para nivel de confianza (1.96 para 95%)
  • • σ = desviación estándar
  • • E = margen de error
  • Error de muestreo: Diferencia entre estadístico muestral y parámetro poblacional
  • Sesgo de selección: Muestra no representativa de la población
  • Ejemplo: Encuesta electoral: no preguntar solo a jóvenes (sesgo)
  • Muestreo sistemático: Seleccionar cada k-ésimo elemento tras un inicio aleatorio
  • • Riesgo de periodicidad oculta en los datos
  • Muestreo por cuotas: No aleatorio; se fijan cuotas por subgrupo poblacional
  • • Más barato pero no permite calcular error de muestreo teórico
  • Error no muestral: Sesgos por formulación, respuesta, o procesamiento de datos
  • • Puede ser más grave que el error de muestreo mismo

  • ¿Qué es? Hacer conclusiones sobre una población a partir de una muestra
  • Intervalos de confianza:
  • • Rango donde probablemente está el valor real
  • • IC 95%: x̄ ± 1.96 × (σ/√n)
  • • "Estamos 95% seguros de que la media real está aquí"
  • Pruebas de hipótesis:
  • • H₀: hipótesis nula (no hay efecto)
  • • H₁: hipótesis alternativa (hay efecto)
  • • p-valor: probabilidad de ver estos datos si H₀ es cierta
  • • Si p < 0.05, rechazamos H₀
  • Errores tipo:
  • • Tipo I: Rechazar H₀ siendo cierta (falso positivo)
  • • Tipo II: No rechazar H₀ siendo falsa (falso negativo)
  • Ejemplo: ¿El nuevo método de enseñanza es mejor?
  • • H₀: no hay diferencia
  • • Si p = 0.03, hay evidencia de que sí es mejor
  • Potencia estadística: Probabilidad de rechazar H₀ cuando es falsa (1 − β)
  • • Depende de tamaño del efecto, tamaño muestral, y nivel de significancia α
  • Intervalo de confianza para diferencia de medias: Comparar dos grupos independientes
  • • Prueba t de dos muestras: t = (x̄₁ − x̄₂) / EE
  • Teorema de límite central aplicado: La distribución muestral de x̄ es aproximadamente normal
  • • Permite construir IC y pruebas sin asumir distribución poblacional específica
  • Valor crítico y región de rechazo: zα/2 define los límites de la zona de rechazo
  • • Prueba unilateral vs bilateral según la dirección de H₁

  • ¿Qué es? Modelar la relación entre dos variables
  • Fórmula: Y = β₀ + β₁X + ε
  • • Y = variable dependiente (respuesta)
  • • X = variable independiente (predictor)
  • • β₀ = intercepto
  • • β₁ = pendiente (cambia Y por unidad de X)
  • • ε = error aleatorio
  • Mínimos cuadrados ordinarios (MCO):
  • • Minimiza la suma de errores al cuadrado
  • • Encuentra la línea que mejor ajusta
  • R² (coeficiente de determinación):
  • • Proporción de varianza de Y explicada por X
  • • Va de 0 a 1 (0% a 100%)
  • • R² = 0.85 significa que X explica 85% de la variación en Y
  • Ejemplo: Relación entre educación (años) e ingresos:
  • • Ingresos = 5000 + 2000 × Años_educación
  • • Cada año extra → $2000 más de ingreso
  • Advertencia: Correlación no implica causalidad
  • Supuestos de MCO: Linealidad, independencia, homocedasticidad, normalidad de errores
  • • Violaciones graves pueden sesgar estimadores e inferencias
  • Prueba t para β₁: H₀: β₁ = 0; verificar si la relación es significativa
  • • t = β̂₁ / EE(β̂₁); se compara con t crítico con n−2 gl
  • Intervalo de predicción: Rango para una nueva observación individual Y₀
  • • Más amplio que el IC de la media porque incorpora el error aleatorio ε

  • ¿Qué es? Modelar Y con múltiples variables independientes
  • Fórmula: Y = β₀ + β₁X₁ + β₂X₂ + ... + βₖXₖ + ε
  • Ejemplo: Ingresos = β₀ + β₁(Educación) + β₂(Experiencia) + β₃(Edad)
  • Coeficientes parciales:
  • • β₁ mide el efecto de X₁ manteniendo las demás constantes
  • Hipótesis sobre coeficientes:
  • • H₀: βⱼ = 0 (la variable no tiene efecto)
  • • H₁: βⱼ ≠ 0 (la variable sí tiene efecto)
  • Problemas comunes:
  • • Multicolinealidad: variables correlacionadas entre sí
  • • Heterocedasticidad: varianza del error no es constante
  • • Omitted variable bias: dejar fuera variables relevantes
  • Selección de variables:
  • • Stepwise, forward, backward selection
  • • Criterios: AIC, BIC, R² ajustado
  • En economía: Se usa para estimar returnos a educación, efecto de política, etc.
  • R² ajustado: Penaliza agregar variables irrelevantes; R̅² = 1 − (SSE/(n−k−1)) / (SST/(n−1))
  • • Siempre menor o igual al R²; mejor para comparar modelos con distinto número de predictores
  • Prueba F global: H₀: β₁ = β₂ = ... = βₖ = 0
  • • Evalúa si el modelo en conjunto tiene poder explicativo significativo
  • Interacción entre variables: Término βₖⱼ(Xₖ·Xⱼ) captura efectos sinérgicos
  • • Ejemplo: el retorno a educación puede ser mayor si hay más experiencia laboral

  • ¿Qué son? Datos ordenados cronológicamente (PIB mensual, inflación anual)
  • Componentes:
  • • Tendencia: dirección a largo plazo (crecimiento del PIB)
  • • Estacionalidad: patrones que se repiten (ventas navideñas)
  • • Ciclo: fluctuaciones alrededor de la tendencia (recesiones)
  • • Ruido: variación aleatoria
  • Estacionariedad:
  • • Media y varianza constantes en el tiempo
  • • Test de Dickey-Fuller para verificar
  • • Si no es estacionaria → primeras diferencias
  • Modelos comunes:
  • • Media móvil (MA): promedio de los últimos k períodos
  • • Autoregresivo (AR): Yt depende de valores pasados
  • • ARIMA: combinación de AR + I (integración) + MA
  • Pronóstico:
  • • Out-of-sample: reservar datos para validar
  • • RMSE, MAE para evaluar precisión
  • Ejemplo: Predecir PIB del próximo trimestre usando datos históricos
  • Función de autocorrelación (ACF) y parcial (PACF): Identifican rezagos relevantes
  • • ACF para identificar MA; PACF para identificar AR
  • Cointegración: Relación de equilibrio a largo plazo entre series no estacionarias
  • • Ejemplo: consumo e ingreso nacional tienden a moverse juntos en el tiempo
  • Modelo ARCH/GARCH: La varianza del error no es constante (volatilidad financiera)
  • • Esencial para modelar riesgos en mercados bursátiles y tasas de cambio
