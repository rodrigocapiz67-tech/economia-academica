1. Límites y Derivadas

+
  • Límite: Valor al que se acerca una función
  • • Ejemplo: lim(x→0) (sen x)/x = 1
  • Derivada: Tasa de cambio instantánea de una función
  • • f'(x) = lim(h→0) [f(x+h) - f(x)] / h
  • Reglas de derivación:
  • • d/dx (xⁿ) = nxⁿ⁻¹
  • • d/dx (eˣ) = eˣ
  • • d/dx (ln x) = 1/x
  • • Regla de la cadena: d/dx [f(g(x))] = f'(g(x)) × g'(x)
  • Aplicaciones económicas:
  • • Costo marginal = d/dq (Costo total)
  • • Utilidad marginal = d/dx (Utilidad total)
  • • Ingreso marginal = d/dq (Ingreso total)
  • Ejemplo: CT = 100 + 20q + 2q² → CM = 20 + 4q
  • Teorema de L'Hôpital: lim(f/g) = lim(f'/g') cuando hay indeterminación
  • Derivadas parciales: ∂f/∂x mide el cambio en f respecto a x manteniendo y constante
  • Diferencial total: df = (∂f/∂x)dx + (∂f/∂y)dy, fundamental para elasticidades

2. Integrales

+
  • ¿Qué es? Operación inversa de la derivada (área bajo la curva)
  • Reglas básicas:
  • • ∫xⁿ dx = xⁿ⁺¹/(n+1) + C
  • • ∫eˣ dx = eˣ + C
  • • ∫(1/x) dx = ln|x| + C
  • Integral definida: Área entre a y b
  • • ∫[a,b] f(x) dx = F(b) - F(a)
  • Aplicaciones económicas:
  • • Ingreso total = ∫(Ingreso marginal) dq
  • • Beneficio = ∫(Beneficio marginal) dq
  • • Superávit del consumidor = área entre demanda y precio
  • Ejemplo: Si CM = 20 + 4q, CT = ∫(20+4q)dq = 20q + 2q² + C
  • Integración por partes: ∫u dv = uv - ∫v du, útil en funciones logarítmicas
  • Integración por sustitución: Cambio de variable para integrales compuestas
  • Teorema del valor medio: Existe c tal que f(c) = (1/(b-a))∫[a,b] f(x) dx

3. Optimización

+
  • ¿Para qué? Encontrar máximos y mínimos (maximizar beneficios, minimizar costos)
  • Condiciones de primer orden:
  • • f'(x) = 0 (punto crítico)
  • Condiciones de segundo orden:
  • • f''(x) > 0 → mínimo
  • • f''(x) < 0 → máximo
  • Ejemplo - Maximizar beneficio:
  • • Ingreso: R(q) = 100q - 2q²
  • • Costo: C(q) = 50 + 20q
  • • Beneficio: π(q) = 80q - 2q² - 50
  • • π'(q) = 80 - 4q = 0 → q* = 20
  • • π''(q) = -4 < 0 → es máximo
  • Método de Lagrange:
  • • Para optimizar con restricciones
  • • L = f(x,y) + λ(g(x,y) - c)
  • • Ejemplo: Maximizar utilidad con presupuesto limitado
  • Matriz Hessiana: H = [[fxx, fxy], [fyx, fyy]] para clasificar puntos críticos en multivariable
  • Condiciones Kuhn-Tucker: Generalización de Lagrange con restricciones de desigualdad
  • Teorema del Envolvente: Cómo cambia el valor óptimo al cambiar parámetros exógenos

4. Álgebra Lineal

+
  • Matrices: Arreglos de números en filas y columnas
  • Operaciones:
  • • Suma: elemento a elemento
  • • Multiplicación: fila × columna
  • • Inversa: A × A⁻¹ = I
  • Sistemas de ecuaciones:
  • • Ax = b → x = A⁻¹b
  • • Ejemplo: 2x + y = 10, x + 3y = 15
  • Valores propios:
  • • Av = λv (escalar λ, vector v)
  • • Importantes en modelos econométricos (PCA)
  • Aplicaciones económicas:
  • • Modelos de input-output de Leontief
  • • Análisis de series de tiempo
  • • Optimización de portafolios
  • Rango y nulidad: Dimensión del espacio columna y del espacio nulo de una matriz
  • Diagonalización: Si A = PDP⁻¹, permite calcular potencias A^k fácilmente
  • Formas cuadráticas: Q(x) = xᵀAx, clave para clasificar puntos críticos en optimización

5. Ecuaciones Diferenciales

+
  • ¿Qué son? Ecuaciones con derivadas de funciones desconocidas
  • EDO de primer orden:
  • • dy/dx = f(x,y)
  • • Solución: y = ∫f(x,y)dx + C
  • Ejemplo - Crecimiento exponencial:
  • • dP/dt = rP (la tasa de crecimiento es proporcional al tamaño)
  • • Solución: P(t) = P₀e^(rt)
  • • P₀ = población inicial, r = tasa de crecimiento
  • Modelo de crecimiento de Solow:
  • • dk/dt = sf(k) - (n+δ)k
  • • k = capital per cápita
  • • s = tasa de ahorro
  • • n = crecimiento poblacional
  • • δ = depreciación
  • Aplicaciones:
  • • Modelos de crecimiento económico
  • • Dinámica de mercados
  • • Difusión de innovaciones
  • EDO de segundo orden: d²y/dx² + a dy/dx + by = f(x), usada en ciclos económicos
  • Diagramas de fase: Representación gráfica de la dinámica del sistema económico
  • Análisis de estabilidad: Determina si el sistema converge a un equilibrio o diverge

6. Análisis de Series de Tiempo

+
  • ¿Qué es? Datos ordenados cronológicamente (PIB mensual, inflación anual)
  • Componentes:
  • • Tendencia: dirección a largo plazo
  • • Estacionalidad: patrones que se repiten
  • • Ciclo: fluctuaciones alrededor de la tendencia
  • • Ruido: variación aleatoria
  • Modelos comunes:
  • • Media móvil: promedio de los últimos k períodos
  • • ARIMA: autoregresivo integrado de media móvil
  • • Exponencial smoothing: más peso a datos recientes
  • Estacionariedad:
  • • Media y varianza constantes en el tiempo
  • • Test de Dickey-Fuller para verificar
  • Ejemplo: Predecir ventas del próximo mes usando datos de los últimos 12 meses
  • Cointegración: Relación de largo plazo entre dos o más series no estacionarias
  • Modelos GARCH: Capturan volatilidad cambiante en el tiempo (riesgo financiero)
  • Función de autocorrelación (ACF): Mide la correlación entre valores separados por k rezagos
